## Project: OctaNet Internship

### ATM Interface

Explore a Python-based ATM simulation offering essential banking operations such as withdrawals, deposits, and transfers. Features include robust user authentication, transaction management, and a streamlined command-line interface (CLI).

- **Key Features**:
  - Secure User Authentication
  - Comprehensive Transaction Management
  - Detailed Transaction History Tracking
  - Intuitive Command-Line Interface (CLI)

- **Skills Developed**:
  - Proficiency in Python Programming
  - Application of Object-Oriented Programming (OOP) Principles
  - Effective Data Management Strategies

- **Future Enhancements**:
  - Implementation of Advanced Security Measures
  - Addition of Enhanced Transactional Capabilities

To get started:
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Execute `python ATM-Interface.py` to launch the ATM simulation.
